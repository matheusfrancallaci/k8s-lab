Kubernetes na Nuvem

Azure Kubernetes Service (AKS) — cluster real para os labs

O cluster gera custo enquanto está ligado. Use Parar ao terminar de estudar — az aks stop pausa o faturamento de compute. Há auto-stop após 45 min de inatividade. Perfil barato: 1 nó standard_d2als_v7, tier Free.

Conectar qualquer cluster — EKS, GKE, OKE, k3s...

Os labs rodam em qualquer cluster presente no seu kubeconfig — a plataforma segue o contexto do kubectl, independente da nuvem. Conecte pelo terminal e o contexto aparece aqui:

AWS EKS

aws eks update-kubeconfig --name meu-cluster --region us-east-1

Google GKE

gcloud container clusters get-credentials meu-cluster --zone us-central1-a

Qualquer outro (k3s, OKE, on-prem remoto...)
